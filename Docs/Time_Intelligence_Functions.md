# DAX Time Intelligence functions (excerpt)

Time intelligence functions let you manipulate and compare values across periods (days, months, quarters, years).

Examples include: `TOTALYTD`, `SAMEPERIODLASTYEAR`, `DATESYTD`, `DATEADD`, `DATESBETWEEN`, `FIRSTDATE`, `LASTDATE`, `STARTOFYEAR`, `ENDOFMONTH`, etc.

Important: many of these functions require a proper Date table to be marked as the model's Date table when using classic time intelligence.

Full list and docs: https://learn.microsoft.com/en-us/dax/time-intelligence-functions-dax
